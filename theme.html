Installing A Theme

Introduction

Turbo offers multiple HTML5 themes that can be easily changed to better suit your projects. Installing a theme is a matter of simply typing in a few terminal commands. When you change themes, the files in the assets folder are replaced and the content of the index.html file is updated with the elements of the theme. Additionally, the Gulp file is updated accordingly with the new assets. If you want to swap out specific things such as images, colors, fonts, etc., you can do so in the assets directory. If you add any additional CSS or JavaScript files, be sure to update your Gulp file and make sure they are properly imported.

Viewing Available Themes

Turbo is constantly being updated with new themes. At any time, you can view which themes are available by typing the following command into your terminal: 

$ turbo themes

Changing Themes

Turbo does not come with a specific theme in the default template. After you have viewed the available themes, you can select a theme by typing the following command into your terminal: 

$ turbo theme THEME_NAME

If you select the hyperspace theme, for example, you would type the following:

$ turbo theme hyperspace

Restart your devserver and refresh your browser. You should see the theme:
